# Recipes

Personal recipe collection in [Cooklang](https://cooklang.org/) format.

## Usage

1. Clone this repo
2. Symlink or sync to iCloud for iOS app access
3. Edit `.cook` files with any text editor

## Structure

```
Category/
  Recipe Name.cook
  images/
    Recipe Name.jpg
```

See [CLAUDE.md](CLAUDE.md) for Cooklang syntax reference.
